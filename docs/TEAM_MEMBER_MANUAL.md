# TEAM MEMBER MANUAL

## Role Overview
The **Team Member** is the core user of the system. They participate in discussions, collaborate on projects, and share files.

## Permissions
- Send and Receive messages in Direct Messages or Groups they have joined.
- Upload and Download files.
- Create private or public groups (depending on organization policy).
- Edit or Delete their own messages.
- Search chat history.

## Workflows

### 1. Collaboration
- Join a Project or Team group.
- Share work-in-progress files for feedback.
- Use message reactions to quickly acknowledge receipt of information.

### 2. Information Retrieval
- Use the global search to find a file shared two weeks ago.
- Review pinned messages in a group to see current priorities.

## Daily Usage Examples
- Sending a message to a colleague to ask for clarification on a task.
- Uploading a daily status report PDF to the Team group.
- Participating in a real-time brainstorming session in a Project group.
