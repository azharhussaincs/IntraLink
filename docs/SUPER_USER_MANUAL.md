# SUPER USER MANUAL

## Role Overview
The **Super User** is typically a Company Owner or Senior Executive. This role focuses on oversight, reporting, and organization-wide communication rather than system administration.

## Permissions
- View all Teams and Groups within the organization.
- View organization-wide reports and analytics.
- Broadcast messages to all users (Organization-Wide Announcements).
- Chat with any user in the system.
- Access project summaries and department productivity reports.

## Workflows

### 1. Reviewing Organization Health
- Access the **Analytics Dashboard**.
- Review daily and weekly productivity reports submitted by Team Leads.
- Monitor file transfer statistics and user activity trends.

### 2. Strategic Communication
- Use the **Broadcast** feature to send a message to every user in the LAN (e.g., "Monthly Town Hall starting in 10 minutes").
- Directly message any Team Lead to discuss department updates.

## Daily Usage Examples
- Reading the "End of Day" consolidated reports from all departments.
- Checking the status of a high-priority project by viewing its specific chat group (Read-only or active participant).
- Verifying overall system adoption through user activity metrics.
