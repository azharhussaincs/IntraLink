# Role Manual: Team Member

## Role Overview
As a **Team Member**, you are the core contributor to the organization. You use CipherLink for daily communication, collaboration, and file sharing.

## Permissions
- Send and Receive Direct Messages.
- Join and participate in Group Chats.
- Upload and Download files of any size.
- View and react to messages.
- Manage your personal profile and settings.

## Key Workflows

### 1. Daily Communication
- Stay logged in to receive real-time updates from your team.
- Use **Direct Messages** for 1-on-1 questions.
- Use **Groups** for team-wide discussions.

### 2. Collaborating on Files
- Upload your work directly to the relevant group.
- Download the latest versions of documents shared by your manager.
- If an upload is interrupted, just click retry to resume.

### 3. Profile Setup
- Add an avatar to help your teammates recognize you.
- Toggle **Dark Mode** if you prefer a darker interface.

## Daily Usage Example
- Sign in and check for any unread messages from the Team Lead.
- Upload a PDF report to the "Daily Sync" group.
- React with a "Thumbs Up" to a colleague's message about a task completion.
