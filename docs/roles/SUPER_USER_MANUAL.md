# Role Manual: Super User (Company Owner)

## Role Overview
The **Super User** represents the organization's top management. You have broad visibility across all departments and teams but do not handle technical system configuration.

## Permissions
- View all Teams and Organization Structure.
- Send organization-wide **Broadcast Messages**.
- Access all **Daily/Weekly Reports** from Team Leads.
- Chat with any user in the organization.
- View high-level project summaries and productivity analytics.

## Key Workflows

### 1. Organization-Wide Announcements
When a company-wide update is needed:
- Use the **Broadcast** tool to send a pinned message to every user.

### 2. Reviewing Reports
- Navigate to the **Reporting Dashboard**.
- Filter reports by Department or Team Lead.
- Export summaries for stakeholder meetings.

### 3. Cross-Departmental Coordination
- Use the **Directory** to find and message any Team Lead or Manager instantly.

## Daily Usage Example
- Review the **Daily Reports** submitted by Team Leads.
- Broadcast an announcement about the upcoming company meeting.
- Consult the **Analytics Dashboard** to see file sharing trends across departments.
