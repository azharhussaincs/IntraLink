# PROJECT MANAGER MANUAL

## Role Overview
The **Project Manager** is role-limited to specific project scopes. They manage the communication and file sharing for particular initiatives.

## Permissions
- Create project-specific chat groups.
- Add/Remove members from project groups.
- Manage project files and documentation.
- Assign tasks (via chat pins or integrated task lists).

## Workflows

### 1. Project Kickoff
- Create a group for the new project (e.g., "Project - Alpha Release").
- Add the required Team Members.
- Upload the Project Charter and Timeline.

### 2. Milestone Tracking
- Use message pinning to highlight current project milestones.
- Request status updates from project members via group chat.

## Daily Usage Examples
- Reviewing CAD files or Source Code shared by members.
- Forwarding project updates to the Team Lead.
- Archiving project groups once the initiative is complete.
